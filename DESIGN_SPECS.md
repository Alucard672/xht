# 乡货通项目 UI 设计示意图

> 本文档提供核心页面的线框图和设计规范，供专业设计师参考进行视觉优化

## 📱 项目概述

**应用类型**: UniApp（微信小程序 + H5）
**目标用户**: 农村批发商家（B端）+ 小卖部老板（C端）
**设计风格**: 类微信原生风格 + 现代化增强
**主题色**: 蓝色 #2d7ff9（专业蓝）

---

## 🎨 设计规范

### 颜色系统

```
主色调:
- 蓝色 #2d7ff9（主要操作、强调）
- 浅蓝 #e8f4ff（背景装饰）

功能色:
- 成功 #07c160 / #34c759（绿色）
- 警告 #ff9500（橙色）
- 危险 #ff3b30 / #ff4d4f（红色）
- 信息 #10aeff（蓝色）

中性色:
- 主文字 #1a1a1a（深黑）
- 次要文字 #666666（中灰）
- 辅助文字 #999999（浅灰）
- 占位文字 #cccccc（淡灰）

背景色:
- 页面背景 #f7f8fa
- 卡片背景 #ffffff
- 输入框背景 #f0f0f5
```

### 字体系统

```
字号层级:
- 特大 48rpx（特大标题）
- 超大 44rpx（超大标题）
- 大 36rpx（大标题）
- 中大 32rpx（小标题/价格）
- 中 30rpx（正文）
- 小 28rpx（次要信息）
- 微小 24rpx（辅助信息）

字重:
- 400 正常
- 500 中等
- 600 半粗
- 700 加粗
- 800 特粗
```

### 间距系统

```
间距单位（以8rpx为基础）:
- 8rpx  (xs)  - 最小间距
- 16rpx (sm)  - 小间距
- 24rpx (md)  - 中间距
- 32rpx (lg)  - 大间距
- 48rpx (xl)  - 超大间距
- 64rpx (xxl) - 特大间距
```

### 圆角系统

```
圆角:
- 4rpx  (sm)  - 小圆角
- 8rpx  (md)  - 中圆角
- 12rpx (lg)  - 大圆角
- 16rpx (xl)  - 卡片圆角
- 50%   (circle) - 圆形
```

## 🎭 交互动效规范

### 页面转场

```
打开页面: 300ms ease-in-out
关闭页面: 250ms ease-in-out
模态弹出: 300ms cubic-bezier(0.4, 0, 0.2, 1)

```

### 设计原则

1. **移动优先**: 所有设计优先考虑移动端体验

3. **信息层级**: 使用字号、颜色、间距区分重要性
4. **一致性**: 统一的视觉语言和交互模式
5. **性能优化**: 减少装饰元素，提升加载速度
