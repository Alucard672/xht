## 🐛 发现问题？

### 快速修复流程

1. **编译错误**
   - 检查错误信息
   - 修复源代码
   - 重新编译: `npm run build:mp-weixin:merchant`

2. **功能错误**
   - 记录问题和复现步骤
   - 检查相关代码
   - 修复并重新编译
   - 再次测试验证

3. **多租户问题**
   - 检查 tenant_id 是否正确传递
   - 检查 Cloud Object 日志
   - 验证数据库查询条件
